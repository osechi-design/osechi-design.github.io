01 Website 架空リフォーム会社サイト

ターゲット

30〜60代の男女（戸建てに住んでいる or リフォームを検討中の人）
家族と暮らしていて、水回りや外壁などの悩みを持っている層

課題

制作ポイント：
● 情報を整理して伝えることが大切
このテーマでは、見出し・段落の使い方や文章構造など、「読み物としてわかりやすいサイト」にする力が求められます。読んでいてストレスのない、整ったページづくりを心がけましょう。
● 「信頼」と「安心」
リフォームは大きな買い物です。ユーザーに「この会社なら任せても大丈夫」と思ってもらえるように、落ち着いた色使い、整ったレイアウト、読みやすい文字サイズ・行間などに気を配りましょう。
● サービスの強みや施工事例をわかりやすく見せる
「どんなことをしてくれる会社か」「実際の事例はどんな内容か」が伝わるように、写真と説明文をセットで配置すると効果的です。見た人が「自分の家もこんなふうにしたい」とイメージしやすくなります。
● お問い合わせへの導線（=CTA）も意識して
「この会社に相談してみよう」と思ったとき、すぐに問い合わせページへ進めるようにしましょう。ページ全体を通して、自然に問い合わせへ誘導できる設計が理想です。

目的

会社の信頼感・実績を伝え、「この会社に相談してみよう」と思ってもらうこと。特に見込み客からの問い合わせを増やすことがゴール。

情報設計

ユーザーが迷わず問い合わせまで到達できるよう、最短3クリックでCV到達可能な構造に整理。
TOPでベネフィット提示
サービス → 施工実績 → お客様の声で信頼醸成
「無料相談」へ誘導（複数導線配置）

コンセプト

「誠実さ ✕ 安心感 ✕ わくわくする暮らしの想像」
白ベース ✕ オレンジのアクセントで“誠実＋前向き”な印象を演出
施工後の暮らしをイメージできる写真中心構成
“任せても大丈夫”と感じるよう人の温度が伝わるUI/写真選定

デザインプロセス

信頼獲得UI設計
◆ 社員紹介・お客様の声に顔写真を掲載し、心理的不安を解消
実績→会社紹介→CTAという“自然な安心型導線” ◆ CV導線最適化
ナビに常時CTA配置+各ページ下部にもCTAを再配置
スクロール後半の離脱前に再度相談導線を設置
◆ UXを意識した視覚演出
Intersection Observerを用いたフェードインで情報を段階的に表示
→ 読み疲れ防止、滞在時間UPを狙う視覚設計

制作期間（８時間実働/日）

企画 / 情報設計 / WF 2日間
デザイン 3日間
コーディング 1週間

使用ツール

Figma / Photoshop / VScode

サイトURL

株式会社わずくらしリフォーム（※架空サイト）

02 Flyer 架空着付け体験教室

ターゲット

20〜40代女性 / 国内旅行客・インバウンド観光客（特に京都観光）
記念日・女子旅・カップル旅で「特別な体験・非日常の写真を残したい層」
SNS映え・和文化体験に価値を感じるユーザー

課題

京都には着物レンタル店が多く、情報が多すぎて差別化がされにくい市場
「着付けだけ」では価値が弱く、フォト体験による付加価値の訴求が必要
多くのチラシが情報羅列型で、世界観・情緒体験が伝わっていない

目的

“着付け＋撮影体験”という物語性を訴求し、予約獲得につなげること
体験前からワクワクさせ、独自の世界観に没入させるブランド設計
SNS投稿を促し、口コミ・認知拡大へ波及する導線を構築

情報設計

体験の“情緒→価値→申込”までを 視線誘導でコントロールする設計
メインビジュアル
→ 非日常体験を一瞬で想起させ、視線を強制的に引き込む
縦書きキャッチコピー
→ 「着付けを学び 京都で残す一枚」=体験の本質価値を7秒で伝達
ストーリー文章
→ “体験の情緒的価値”を言語化し、価格ではなく感情で動かす
プラン情報 & QRコード
→ 右下に集約し、即アクションへ導線を作る（Instagram/公式サイト）

デザインプロセス

世界観設計：秋の京都×和の静寂を基調に、黒×金×紅葉色で高級感と余白の余韻を演出
フォト選定：背景の深度と和傘の色を活かし、視線が人物に向かう構図を採用
フォント設計：縦書き明朝で“伝統の品格”、本文は可読性重視のゴシックでバランス調整
レイヤー処理：黒のグラデーションを下部に敷き、文字の視認性を担保
アイコン配置：QRコードとSNS導線は“世界観を壊さない最小限デザイン”で統一

制作期間（８時間実働/日）

約◯日（企画構成 → ラフ制作 → 本制作 → 色調補正 → 入稿データ作成）

使用ツール

Adobe Illustrator / Photoshop / Figma

03 Logo Nasuko.工房様デザイン提案

ターゲット

現在顧客は40代以上の女性が中心。ただし、ターゲットは特に絞っていない。
手作りやアート性に価値を感じる、感性重視の来場者
「作家本人の想いや世界観に共鳴して作品を迎えたい」人
マルシェ・ハンドメイドイベント・クラフト市の来場者（20〜50代）

課題

一点もの中心のアート性の高い作品だが、遠目から世界観が伝わりにくく埋もれやすい
作品の哲学や温度感が、短時間の接客では伝えきれない
認知度が低く、SNS導線が弱いため「再会」が生まれないこのブランドは「商品を売る」のではなく、 “作品とのご縁を届ける場所” を目指している。
すべて一点もの。大量生産はせず、作家自身が心から納得した作品のみを販売
作品には物語が宿り、手に取った人の心にふっと灯がともるような温度感を大切に
誰かの生活の中で、“時代を越えて愛される小さな相棒”になることを願って制作
お客様との出会いを大切にし、帰り道に「また会いに行きたい」と思われるお店を目指す
作者の“作品観＝人生観”を、ブランドとして視覚化することが今回のデザインの軸。

目的

この度のプロジェクトは、陶芸作品を制作・販売されている「Nasuko.工房」様のブランディングツールを一新するものです。
クライアントからのヒアリングシートに基づき、以下の制作物についてデザインをお願いします。
温かみのある、手作りの良さが伝わるデザインを目指します。
1. ショップカード制作
目的: イベント出店時や作品購入者にお渡しし、お店の認知度向上と再来店（インスタグラムへのアクセス）を促す。
2. ロゴ制作
目的: お店の「顔」となるシンボルを作成し、ショップカードや看板など、今後のブランディング展開の核とする。
遠くからでも一瞬で世界観が伝わる屋号看板の制作
作品の“温度・ユーモア・物語性”を視覚的に表現し、ブースの前で足を止めさせる
SNSを強制ではなく**“自然と繋がりたくなる形”**で導線を作り、再会率を高める

デザインプロセス

● ロゴ（物語性 ✕ 会話が生まれる仕掛け）
現在のInstagramキャラクターを再解釈し、温度と余白を残す線画に再構築
瀬戸市（産地）＝椿モチーフを忍ばせ、「出身どちらなんですか？」と**自然に会話が生まれる“種”**を設計
● ショップカード（作品を迎える儀式のような体験設計）
購入者が“持っていたくなるカード”を目指し、アートカードのようにデザイン
クラフト紙×深いブラウンで、土・火・手仕事のぬくもりを視覚化
裏面はコピーを詩的に添え、フォローの導線が“強要ではなく共鳴”で生まれる構成
● 屋号看板 / タペストリー（世界観の“幕開け”）
木材 ✕ ウッドバーニングで手仕事の痕跡が感じられる質感
ロゴを大きく配置し、マルシェでも遠距離視認性を担保
タペストリーは“撮りたくなる背景”として設計し、投稿されやすい設計に

制作期間

約◯日（ヒアリング → 世界観抽出 → ロゴ・コピー開発 → ツール展開）

04 Pop 株式会社E-es-E様ブランドコンセプト提案

ターゲット

25〜35歳 女性（共働き・子育て層）
コスパと生活快適性を重視、だけど“エコ意識は持っていたい”層
「説教臭くない、気軽で楽しいエコ発信」に反応するタイプ

課題

商品自体の魅力や差別化ポイントは強いが、認知拡大とSNS発信が弱い
“エコ疲れ（サステナ疲れ）”により、若年層に刺さりにくい市場背景
社長自らデザイン制作が可能なため、外部デザイナーが入る価値の明確化が必要

目的

若者に刺さるブランド世界観を再構築し、商品との出会いを楽しい体験に変えること
“気軽に続けたくなるエコ”というコンセプトで、ポジティブな印象形成
SNS上で拡散・共感される仕掛けをつくり、ブランド認知を底上げ

戦略

施策 狙い
市場調査 ✕ ターゲットインサイト分析 “刺さる切り口”を科学的に抽出
キャラクター開発 世界観の象徴として認知と共感を獲得
POPデザイン 売り場で一瞬で目に留まり、記憶に残す
SNS展開（Instagram） キャラを媒体として継続接点を強化

コンセプト

「気楽 ✕ エコ ✕ エンタメ」
エコを“正しさ”で押し付けるのではなく、
思わず笑って、気楽に続けられる世界観を採用。
“楽しさから入るエコ”をデザインし、行動ハードルを下げるブランドへ。

キャラクターデザイン「ホダンディ」（手書き制作）

洒落好きな社長のキャラ性を投影し、親しみとユーモアを両立
昭和レトロ ✕ ダンディ ✕ エコという“ズラし”で若者の共感を狙う
北の深海からやってきた“地球を磨く紳士”というストーリーを設定
キャラクター人格が語る形式で**“説教しないエコ啓発”**を実現
（例：#ホダンディ語録 #自然に任せたほうが、うまくいくこともある）

POPデザイン（売り場での“瞬間突破力”）

若者が思わずスマホで撮りたくなるデザイントーンを採用
キャラクターをアイキャッチにし、秒で“世界観→商品特徴”が伝わる構成
文字量を最小化し、視認性と笑顔の誘発を最優先

SNS展開戦略（世界観で繋ぐファン化導線）

ホダンディが語る10秒投稿で、商品情報 ✕ エコ知識 ✕ 世界観を発信
UGC（一般ユーザー投稿）を誘発する #ホダンディチャレンジ を企画
“世界観が楽しいから続く”エコブランドとして継続接点を構築

制作期間

約◯日（市場調査 → 戦略設計 → キャラクター制作 → POPデザイン → SNS企画）

05 Sticker オリジナルステッカー制作

制作背景

Illustrator習得のアウトプットとして、自主企画で制作。
訓練校で出会ったクリエイター仲間からのアドバイス・ブラッシュアップを通し、**“感謝を形に残す作品”**として完成。

コンセプト

「子どもとの日常を、持ち歩ける可愛い存在に」
一般的な“BABY IN CAR”は貼る場所が限定的。
「HELLO, I’M HERE!（ここにいるよ）」という言葉に変えることで、車以外のあらゆる場所で使える汎用性のあるデザインへ再解釈。
用途の視野を広げ、持ち物に“自分の存在”を添えるステッカーとして企画。

デザインポイント

6cm ✕ 6cmのスクエア設計
→ PC・スマホケース・水筒など、日常の持ち物に馴染むサイズ感
サンセットグラデーション
→ トレンドを取り入れつつ、作者の“好きな景色（夕暮れの海）”を反映
遊び心ある2種展開
HELLO, I’M HERE!：存在を可愛くアピール
PEEK-A-BOO!：いないいないばあ！の遊び心で親しみを演出
素材選定
マット仕上げ・防水・耐候性で、**安っぽく見えない“グッズ品質”**に

ターゲット

小さなお子様のいる家庭
子どもモチーフの雑貨が好きな人
PC・文房具・ボトル等にステッカーで自己表現したい層
クリエイター交流会で“名刺代わりに使えるデザインツール”として活用

活用シーン

クリエイター交流会・イベントで配布し、自己発信ツールとして活用
裏面にInstagram QRを設置し、**名刺以上の“記憶に残る接点”**を作る設計
配布後のSNSフォロー率が通常の名刺より高い手応えあり

今後の展開

SUZURIでの物販展開を予定（キーホルダー/iPhoneケース 等）
キャラクターシリーズ化（子ども→大人→ペット展開）
“HELLO, I’M HERE!” をママギフトブランド化する構想あり

制作期間

約◯日（ラフ案 → デザイン制作 → 改良 → 印刷検証）

使用ツール

Adobe Illustrator

06 Illust イラスト(Instagram)

子育てイラスト

Instagram投稿

制作背景

元々イラストが好きだったが、一歩踏み出せずにいた中、職業訓練校で“育児イラスト投稿者”と出会い挑戦を開始。
「自分の経験が誰かの心を軽くできるなら」という想いで、子育て世代に寄り添うイラスト発信をスタート。
7月〜10月の4ヶ月間で 42投稿を継続達成。
趣味として始めた活動が、イラスト表現 ✕ SNS運用 ✕ ユーザー反応分析を実践。

コンセプト

“日常の一瞬を、クスッと笑える物語に。”
子育ては幸せでも、きれいごとだけではない。
その「リアル・笑い・発見・感動」を切り取り、パパ・ママの心が少し軽くなるイラストを目指している。
子育てのリアルを、可愛さ・笑い・気づきに変換し、**心がふっと軽くなる“共感アート”**として届ける。

ターゲット

0〜6歳の子どもを育てるパパ・ママ
忙しい育児の合間に、癒しや共感を求めてSNSを見る層
自分と同じ境遇の発信者に“仲間意識”を感じたいユーザー
今後は希少性の高い **“パパ視点で描く育児イラスト”**で差別化を狙う。

投稿テーマ

子どもの発想・言動の可愛さ
家庭の“あるある”＆失敗談
育児の大変さと、それでも愛おしい瞬間
日常の尊さに気づく “スローな時間軸”
見る人の「わかる!」を生む“共感3秒設計”を意識。

マーケティング実験（Instagram広告 ✕ 反応分析）

デザイン ✕ SNSマーケを学ぶため、自主的に広告配信を実施。
異なるテイストの投稿を広告に出し、反応差を分析。
検証した変数：
タッチの違い（柔らかい水彩／線画／ストーリー漫画風）
文字あり / 文字なし
子ども単体 / 親子構図
結果、**“共感系ストーリー＋余白のある絵”**が保存率とフォロー転換が高い傾向。
→ 自分の強みを把握し、今後の方向性（パパ視点の物語型）を確立する材料に。

制制作のこだわり

柔らかい線と淡い色で “安心感” と “親しみ” を表現
写真の再現ではなく、“記憶の温度が蘇るタッチ”
余白を活かし、ユーザーが自分の経験を重ねられる構成

得た学び

継続投稿により習慣化＆アウトプットスピード向上
SNSは絵の上手さより**“共感設計”が鍵**であると体感
ユーザー反応を見て改善する PDCA型クリエイティブの重要性を実感

今後の展望

2025年内の収益化
SUZURI等でグッズ化
note ✕ インスタで育児エッセイ連載
X/TikTokで“パパ視点育児”を横展開
パパ×育児×クリエイティブという“独自領域”の確立
